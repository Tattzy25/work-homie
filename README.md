# Acme Inc. - Content Creator Platform

A comprehensive platform for content creators featuring AI-powered tools for trend analysis, content planning, viral calendar management, and creator workflows.

## Features

- **Dashboard**: Overview of your content performance and analytics
- **Trend Pilot**: AI-powered trend discovery and analysis
- **Content Analysis**: Deep dive into content performance metrics
- **Content Calendar**: Viral calendar planning and scheduling
- **Creator Network**: Connect with other creators
- **Trend Oracle**: Metadata and tagging tools
- **Video Editor Pro**: Professional video editing capabilities
- **Creator Workflow**: Streamlined content creation workflows
- **Automation Studio**: AI-powered automation tools

## Tech Stack

- **Framework**: Next.js 16
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI
- **Icons**: Tabler Icons
- **State Management**: React Hooks
- **Charts**: Recharts
- **Forms**: React Hook Form with Zod validation

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Tattzy25/work-homie.git
   cd workshop
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Run the development server:
   ```bash
   pnpm dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
